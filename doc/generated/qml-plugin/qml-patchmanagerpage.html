Detailed Description

Page shown through the Jolla Settings Plugin from the Settings Application.

It is a both a front-end to the Daemon and a Patch management application.

It allows to:

  • View the list of installed and activated Patches.
  • Configure the Application and Daemon settings.
  • Activate or deactivate Patches
  • Install Patches from the Web Catalog
  • Uninstall installed Patches

Method Documentation

function showUpdates(manual)

Flashes the Pulley Menu if updates are available and manual is false. Does nothing if manual is true.
