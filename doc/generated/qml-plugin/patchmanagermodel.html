Detailed Description

A ListModel containing the metadata of Patches.

Currently it defines the following Roles:

Qt RoleQML Role Name
Qt::DisplayRolename
Qt::DecorationRolesection
Qt::EditRolepatchObject

See also Qt::QAbstractItemModel.

void PatchManagerModel::populateData(const QVariantList &data, const QString &patch, bool installed)

Does nothing if both data and patch are empty.

If patch is empty, and we have data, extracts the patch name from data and adds the metadata to the model.

If we have patch and data, and installed is true, add the data to the model.

If we have patch and data, and installed is false, just create a PatchObject from data.

Warning: that last part is not yet documented.

See also saveLayout().
