Detailed Description

The WebPatchesModel holds elements from the Web Catalog.

Roles follow the Patch JSON format, hence the following are defined:

Qt RoleQML Role Name
description
last_updated
name
display_name
category
author
rating
total_activations

See also Qt::QAbstractItemModel.
